# QurbanCare – Berbagi di Hari yang Mulia

**QurbanCare** adalah aplikasi manajemen ibadah kurban berbasis Android yang dirancang untuk mempermudah koordinasi antara pekurban, panitia, dan penerima manfaat. Aplikasi ini mengintegrasikan sistem manajemen hewan, pelacakan distribusi, dan transparansi data dalam satu platform yang user-friendly.

## 🚀 Fitur Utama
Aplikasi ini mencakup beberapa modul manajemen utama:
* **Manajemen Hewan Qurban**: Memantau data sapi dan kambing (berat, umur, jenis kelamin, dan harga).
* **Sistem Slot Kurban**: Fitur pengambilan slot kurban dengan progres pembiayaan yang transparan.
* **Manajemen Panitia & Peserta**: Pendataan kontak panitia serta status pembayaran peserta kurban.
* **Distribusi & Penerima**: Pengelolaan data penerima manfaat (Warga, Dhuafa, Pesantren) dan status pengambilan daging.
* **Laporan Keuangan**: Pencatatan transaksi masuk, upload bukti pembayaran, dan status pelunasan.
* **Tanya QurbanCare AI**: Asisten pintar berbasis AI untuk menjawab informasi seputar kurban.
* **Monitor Keramaian**: Integrasi kamera untuk memantau kepadatan di lokasi penyembelihan.

## 🎨 Spesifikasi Desain
Berdasarkan standar UI/UX yang telah ditetapkan dalam prototipe:
* **Tipografi**: Menggunakan font **Poppins** untuk semua elemen teks.
* **Palet Warna**:
    * `#1E7A4C` / `#2FA36B` (Warna Primer - Hijau Islami)
    * `#F4C542` (Warna Aksen - Kuning Emas)
    * `#F5F1E8` (Warna Latar Konten)
* **Komponen**: Penggunaan *Card Rectangle* dengan *Corner Radius* 20 untuk estetika modern.

## 🛠 Teknologi yang Digunakan
* **Platform**: Android Studio
* **Bahasa**: Java
* **Database**: Google Firebase / Firestore (Real-time data)
* **AI Engine**: Groq API (ChatAiActivity)

## 📱 Antarmuka Pengguna (UI)
* **Splash Screen**: Menampilkan pesan "Berbagi di Hari yang Mulia".
* **Dashboard**: Menu navigasi cepat ke fitur Hewan, Peserta, dan Distribusi.
* **Data Management**: Tabel dan list yang bersih untuk memantau status "Sudah Diambil" pada modul distribusi.
